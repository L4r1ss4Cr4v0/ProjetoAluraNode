Sobre o projeto

A ideia dessa API se derivou da imersão com a Alura + Google sobre Node.js. Durante as aulas fomos aprendedosndo sobre o framework, Javascript e MongoDB.

Fizemos um projeto para auxiliar na compreeção do conteúdo: uma API que fornece dados e imagens.

O projeto está armazenado no Google Clound, além de conter uma integração com a IA do Google (Gemini) que é a responsável por gerar a string de descrição da imagem. Todas as imagens são feitas pela Gemini.

Modo de usar

Retorna um post aleatório em formato JSON

http://localhost:3000/posts/random

Exemplo de resposta

{
"_id" : "674640a325dc93dc668b1ed6",
"descricao" : "Rua portuguesa com vista deslumbrante da Torre Eiffel.",
"imgUrl" : "http://localhost:3000/674640a325dc93dc668b1ed6.png",
"alt" : "Imagem da Torre Eiffel em Portugal"
}

Retorna todos os posts

http://localhost:3000/posts

Galeria

A API ofereçe mais de 30 imagens da Torre EIffel em diferentes lugares do mundo, cada uma contendo sua descrição customizada pelo Gemini
